Los componentes incluidos en csvstorage-src.zip se enumeran a continuación:

	- fuentes: Contiene el código fuente de la aplicación:
		- csvstorage

	- lib: Librerías que usa la aplicación CSVStorage
	- resources
		
		- config
			- schemas: Esquemas XSD de CSVQueryDocument y Documento ENI
			- admin.properties
			- csvstorage.properties
			- datasource.properties												
			- dir3.properties
			- eeutil.properties
			- ginside.properties
			- log4j.properties
			- repository.properties
			- trutstore.jks
			- ws-security.properties

		- external-libs: librerías externas y necesarias de instalar manualmente
			- bigDataTransfer-1.0.jar
			- install_readme.txt
		
		- scripts_bbdd: Scripts de BBDD
			- BDcsvstorage.sql
	
	- Web Services externos:

		- dir3: Especificación de WS de obtención de Unidades Orgánicas de DIR3 (WSDL).
		- eeutil: Especificación de WS para validación de Documentos ENI (WSDL).
		- inside: Especificación de WS de creación de Documentos ENI (WSDL).
		 
		
	- licenses: Contiene las licencias bajo las que se distribuyen las librerías de las que depende CSVStorage.

	- LICENSE_EUPL_1.1.pdf: Licencia EUPL1.1, bajo la que se distribuye Inside.
